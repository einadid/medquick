# QuickMed - Pharmacy Management & E-Commerce System

[![License](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![PHP Version](https://img.shields.io/badge/PHP-7.4%2B-blue.svg)](https://php.net/)
[![MySQL Version](https://img.shields.io/badge/MySQL-5.7%2B-blue.svg)](https://www.mysql.com/)

QuickMed is a modern, multi-functional pharmacy management and e-commerce platform designed to streamline medicine sales, inventory management, and multi-shop operations.

## Features

- **User Management**
  - Customer registration and profile management
  - Admin and salesman roles with different permissions
  - Secure authentication system

- **Inventory Management**
  - Medicine stock tracking
  - Expiry date monitoring
  - Low stock alerts
  - Barcode support

- **E-Commerce**
  - Product catalog with search and filters
  - Shopping cart and checkout
  - Order tracking
  - Multiple payment methods

- **Reporting & Analytics**
  - Sales reports
  - Inventory reports
  - Customer analytics
  - Export to Excel/PDF

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Apache/Nginx web server
- Composer
- Node.js (for frontend assets)
